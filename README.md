# PocketJS
Definitely not inspired by that monster-catching game from the late '90s. Built from scratch with HTML5 Canvas and vanilla JavaScript.

A small browser RPG I've wanted to build for a while.

I grew up playing way too many classic monster-catching games, so this is my excuse to learn game development by recreating that feeling from scratch with HTML5 Canvas and vanilla JavaScript. No game engine, just code, trial and error, and lots of console logs.

The goal isn't to make a clone. It's to understand how these games work behind the scenes, one system at a time.

> 🚧 Still a work in progress.

## What's in it

- Tile-based movement
- Turn-based battles
- NPC interactions
- Inventory system
- Save/load progress
- Sprite animations

## Built with

- JavaScript
- HTML5 Canvas
- CSS

## Running locally

```bash
git clone https://github.com/realjyce/pixel-monsters.git
cd pixel-monsters
```

Open `index.html`, or run a local server.

```bash
python -m http.server
```

## Why?

Mostly because it sounded fun.

I've always been curious about how old-school RPGs handled movement, collisions, encounters, dialogue, and battles. This project is my playground for figuring those things out without hiding behind a framework.

Hopefully it ends up being something other people enjoy poking around in too.
